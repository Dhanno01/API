# User Management API

This is a simple Express.js application that provides a RESTful API for managing users. The application supports retrieving user information, adding new users, updating existing users, and deleting users. User data is stored in a JSON file (`MOCK_DATA.json`).

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v12 or higher)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/yourusername/user-management-api.git
   cd user-management-api

2. Install dependencies:

   npm install

## API Endpoints

   GET All Users : '/api/users'

   GET User By ID : '/api/users/:id'

   POST New User : '/api/users'

   PATCH User By User : '/api/users/:id'

   DELETE User By User : '/api/users/:id'

   Get Users as HTML : '/users'
